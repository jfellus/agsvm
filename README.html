GOSSIP BASED DECENTRALIZED MULTIMEDIA INDEXING

Compiling

  1. cmake .
  2. make

System configuration

If you plan to use Shared Matrices functionalities, first check that your kernel allows large enough shared memory segments :
ipcs -l If needed, increase allowed limits by :
  1. Appending the following lines to your /etc/sysctl.conf 
    ###############################
# RETIN SHARED MEMORY
# more info : http://pic.dhe.ibm.com/infocenter/soliddb/v6r5/index.jsp?topic=/com.ibm.swg.im.soliddb.accel.doc/doc/sma.parameters.modifying.overview.html

# Max nb of shm segments
kernel.shmmni=4096 
# Max size of a shm segment 
kernel.shmmax=3000000000
# Max total shm size 
kernel.shmall=4000000000
  2. typing sysctl -p to register these changes
